给定一个整数 n，求以 1 … n 为节点组成的二叉搜索树有多少种？
动态规划

  • 初始化

    假设n个节点存在二叉搜索树的个数是dp（n）,令f(i) 为以i为根的二叉搜索树的个数，则有

               dp(n) = f(1) + f(2) + f(3) +... + f(n)

    当i为根节点时，其左子树的个数为(i - 1),右子树的个数为（n-i）,则

               f(i) = dp(i-1) *dp(n-i)

    结合两个式子可以得到卡特兰数：

               dp(n) = dp(0) *dp(n-1) +dp(1) *dp(n-2) +... +dp(n-1)*dp(0)

                    = dp(n-1) * C(2n,n)

                    = dp(n-1) * (4*n-2)/(n-1)
